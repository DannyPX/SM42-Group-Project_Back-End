| ------- | ------------ |

| Request                       | Description         |
| ----------------------------- | ------------------- |

| `PUT /api/user`               | Update a user       |
| `DELETE /api/user{params}`    | Delete a user       |
| `GET /api/user{params}`       | Get one user        |
| `POST /api/user/auth{params}` | Authenticate a user |
